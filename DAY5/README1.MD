Recursive Function :- When a function calls itself again and again , such type of function is called a recursive function. and the process is called recursion.

| Feature              | **Loop**                                                       | **Recursion**                                            |
| -------------------- | -------------------------------------------------------------- | -------------------------------------------------------- |
| **Meaning**          | Statement jo baar-baar execute hota hai (for, while, do-while) | Function jo khud ko hi call karta hai                    |
| **How it works**     | Repetition using **iteration**                                 | Repetition using **function calls**                      |
| **Speed**            | Fast (no extra function calls)                                 | Slower (every call uses extra memory)                    |
| **Memory Use**       | Less memory use                                                | More memory use (stack frame create hota hai)            |
| **Ending Condition** | Loop condition false hoti hai                                  | Base case milne par recursion rukta hai                  |
| **Use Cases**        | Simple repetitive tasks (sum, table, printing numbers)         | Tree/graph problems, factorial, Fibonacci, complex logic |
| **Readability**      | Easy to understand                                             | Sometimes hard but clean code for some problems          |
| **Example**          | for loop, while loop                                           | factorial, Fibonacci, tree traversal                     |
