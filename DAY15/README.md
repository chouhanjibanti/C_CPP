Friend Function :- 

A friend function is not a member of a class , but by using this we can access the private and protected data of the class.

Key Points :- 
1. Used to access the private data of a class from outside.
2. Declared inside the class using the friend keryword.
3. Recrives object as a paramter .
4. Called normally ,not using the object.
