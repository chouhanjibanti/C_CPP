Constructor :- 
Constructor is a special function inside a class.
It is automatically called when object is created.
constructor is always declared in public section.

Types of Constructor :- 
1. Default Constructor (no paramterized Constructor)
2. Paramterized Constructor
3. copy Constructor



==================================

Copy Constructor :- 
A copy consturctor is used to copy the value of the one object into another object.
