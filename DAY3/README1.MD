Diff types of Loop :- 
1. for loop 
2. While loop
3. do-While loop

Diff between for and while loop 
1. In for loop number of iterations is known.
1. In while loop number of iterations is unkonwn.

2. In for loop initiltization ,condition and incre/decre in single line.
2. In While loop init , condition and incre/decre in diff diff line.

3. Syntx :- 
for(init; condition;incre/decr){
     // statement 
}

3. Syntax :-
init 
while(condition){
    // statement 
    // incre /decre
}



Diff between while loop and do-while loop
1. It is Entry control loop.
1. It is Exit control loop.

2. It will run based on the condition.
2. It will run once .[atleast one time run]

3. Syntax :-
init 
while(condition){
    // statement 
    // incre /decre
}

3. Syntax :- 
init
do{
    //statement
    // incre/decre
}while(condition)
