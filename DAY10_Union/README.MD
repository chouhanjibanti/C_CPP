Union :- 

A union is user defined data type that allows different data member/variable to share the same  memory location.

Note :- Only one member can store value at a time.


Syntax :- 
union union_name{
    datatype member1;// char name[20]  []
    datatype member2; // int age;
    .........
    ........
}
==================================================



Pointer :- A pointer is a variable store the addesss of the another variable.

* -> value at that address(dereference operator)
& -> Address of variable
